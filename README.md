# adrielgama.dev [![GitHub Repo stars](https://img.shields.io/github/stars/adrielgama/adrielgama.dev?style=social)](https://github.com/adrielgama/adrielgama.dev/stargazers)

Personal portfolio of **Adriel de Oliveira Gama**, built with Next.js, [shadcn/ui](https://ui.shadcn.com/), and [scrollx ui](https://scrollxui.dev/).

# Features

- Built using Next.js 16, React, Typescript, Shadcn/UI, TailwindCSS, Motion, ScrollX UI
- Includes sound effects
- Responsive for different devices
- Optimized for Next.js and Vercel

# Getting Started Locally

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/adrielgama/adrielgama.dev
   ```

2. Move to the cloned directory

   ```bash
   cd adrielgama.dev
   ```

3. Install dependencies:

   ```bash
   pnpm install
   ```

4. Start the local Server:

   ```bash
   pnpm dev
   ```
